# Eólica Naranco S.L. - Sistema de monitorización

App Node.js del sistema de monitorización del parque eólico de la sierra del Aramo.

## Instrucciones

Construye la imagen y arranca el contenedor. Algo no funciona, usa los logs para diagnosticar.

## Estructura

- `app.js` — servidor principal
- `Dockerfile` — configuración del contenedor
